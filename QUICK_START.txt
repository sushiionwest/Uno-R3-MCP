ARDUINO MCP SERVER - QUICK START GUIDE
================================================================================

Your Arduino MCP server is installed and tested!
Your Arduino is detected on: COM3

NEXT STEPS:
-----------

1. CONFIGURE CLAUDE CODE
   
   Edit this file:
   %APPDATA%\Claude Code\claude_desktop_config.json
   
   Add this content:
   {
     "mcpServers": {
       "arduino": {
         "command": "node",
         "args": ["C:\arduino-mcp-server\index.js"]
       }
     }
   }

2. RESTART CLAUDE CODE
   
   Completely close and reopen Claude Code

3. UPLOAD ARDUINO SKETCH
   
   Open: examples/thrust_test.ino
   Upload to your Elegoo Uno R3
   
   (Or use your own sketch that sends data via Serial)

4. TEST IT
   
   Ask Claude Code:
   "List all serial ports and connect to my Arduino"
